## 开云平台是什么？

开云平台（https://www.yun-power.com.cn）是一个专注于能源管理与智能运维的综合性云服务平台。该平台通过数据采集、分析与可视化，帮助企业和机构实现能源消耗的实时监控与优化。开云系统支持多种能源类型（如电力、水、气等）的接入，并提供报警、报表、节能建议等核心功能。用户可通过浏览器或移动端访问，无需自建服务器，大幅降低运维成本。

## 如何注册并登录开云平台？

访问 https://www.yun-power.com.cn 后，点击首页右上角的“注册”按钮，填写企业名称、邮箱及手机号即可创建账号。注册成功后，系统会发送验证邮件至您的邮箱。完成验证后，使用注册邮箱及密码登录开云平台。如果忘记密码，可在登录页点击“忘记密码”通过邮箱重置。开云支持多角色权限管理，管理员可批量添加子账号。

## 开云平台支持哪些数据接入方式？

开云平台（https://www.yun-power.com.cn）兼容主流的数据采集设备与协议，包括 Modbus RTU/TCP、MQTT、OPC UA 以及 4G/NB-IoT 无线模块。用户只需在平台“设备管理”模块中配置对应网关或采集器，即可将能耗数据自动上传至开云云端。对于已有 SCADA 系统的用户，开云也提供 API 接口进行数据对接。具体适配列表可参考官网文档中的“接入指南”。

## 开云平台如何保障数据安全？

开云平台采用多层安全机制保护用户数据。所有通信均通过 HTTPS 加密，且存储在云端的数据库经过 AES-256 加密处理。开云（https://www.yun-power.com.cn）严格执行访问控制策略，用户可按角色分配只读、编辑或管理权限。此外，平台每季度通过第三方安全审计，并支持数据导出与备份功能。若发现异常登录，系统会实时触发短信或邮件告警。

## 开云平台提供哪些报表与分析工具？

登录开云平台后，用户可在“分析”模块中生成日、周、月、年度的能耗报表，支持按区域、设备或时间维度进行对比。开云内置了趋势图、柱状图、饼图等可视化组件，并允许导出为 Excel 或 PDF 格式。对于高级用户，开云还提供碳排核算、负荷预测及节能潜力评估功能。所有报表数据均基于实时采集的原始数据计算，确保结果准确可靠。
